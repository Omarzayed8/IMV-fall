Process book

This storybook represents my trip from Toronto to Montreal
The first I wanted to represent the key 2 cities which is Toronto and Montreal
I used Figma to draw the skyline of the 2 cities
Then I draw the major train stops Union Station and Gare Central
I wanted to add some visual elements such as the stars and towers using shapes
I used Yellow colour as a negviation tool
I used a Figma plugin to convert the drawings into a html code
I struggled a bit with hyperlinking objects , then it made me understand the importance of the different tags used
The last page of the story book , I drew a croissant using shapes and strokes
